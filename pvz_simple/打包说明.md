# 游戏打包说明

## 方法一：使用批处理文件（最简单）

直接双击 `启动游戏.bat` 即可运行游戏。

## 方法二：使用 Python 窗口程序（推荐）

双击 `启动游戏.pyw` 运行（不会显示命令行窗口）。

## 方法三：打包成独立 EXE 程序（最精美）

### 步骤：

1. **安装 PyInstaller**
   ```bash
   pip install pyinstaller
   ```

2. **创建图标（可选）**
   ```bash
   pip install Pillow
   python create_icon.py
   ```
   这会生成 `game_icon.ico` 图标文件。

3. **打包程序**
   ```bash
   pyinstaller --onefile --windowed --icon=game_icon.ico --name="植物大战僵尸" main.py
   ```
   
   或者如果没有图标：
   ```bash
   pyinstaller --onefile --windowed --name="植物大战僵尸" main.py
   ```

4. **运行打包后的程序**
   打包完成后，在 `dist` 文件夹中会生成 `植物大战僵尸.exe`，双击即可运行。

### PyInstaller 参数说明：
- `--onefile`: 打包成单个 exe 文件
- `--windowed`: 不显示命令行窗口（GUI 模式）
- `--icon=game_icon.ico`: 指定程序图标
- `--name="植物大战僵尸"`: 指定生成的 exe 文件名
