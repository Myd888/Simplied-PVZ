# 植物大战僵尸简化版

一个使用 Pygame 开发的简化版植物大战僵尸游戏，适合学习和练习游戏开发。

## 安装

### 1. 创建 conda 环境（推荐）

```bash
conda create -n cursor_game python=3.11 -y
conda activate cursor_game
```

### 2. 安装依赖

```bash
pip install pygame
```

## 运行

### 方法一：直接运行可执行文件（最简单，推荐）

- 下载 `Simplified_PVZ.exe` 文件
- 双击运行即可，无需安装 Python 或任何依赖

### 方法二：使用启动程序

- **Windows**: 双击 `启动游戏.pyw`（不显示命令行窗口）
- **或**: 双击 `启动游戏.bat`（显示命令行窗口）

### 方法三：命令行运行

```bash
python main.py
```

## 操作说明

### 基本操作

- **种植植物**: 左键点击网格格子
- **移除植物**: 点击右上角铲子图标，然后点击要移除的植物
- **取消铲子**: 选中铲子后，点击铲子下方的棕色框

### 快捷键

- **空格键**: 暂停/继续游戏
- **Q键**: 退出游戏（会弹出确认窗口）

### 游戏规则

- 僵尸会从右侧随机生成，向左移动
- 植物会自动检测所在行的僵尸并发射子弹
- 子弹命中僵尸会造成伤害
- 僵尸碰到植物会攻击植物
- 僵尸到达屏幕左侧则游戏结束
